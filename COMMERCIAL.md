# WANDO → продукт: коммерческая стратегия (v1, 2026-07-10)

## Позиционирование — одно предложение

**WANDO — командный центр фаундера, который говорит, что делать прямо сейчас, по всем твоим проектам сразу.**
Jira/Asana/ClickUp — системы УЧЁТА работы для команд. WANDO — система РЕШЕНИЙ для владельца.

## Почему уйдут с Jira / Asana / Trello / ClickUp / Basecamp

Не уйдут. И не надо — в этом клин:

| Они | WANDO |
|---|---|
| Единица = задача/доска ОДНОГО продукта | Единица = «твой следующий шаг» по ВСЕЙ империи проектов |
| Ты приходишь смотреть на дашборд | ИИ сам пингует тебя в Telegram (и не задалбывает: 48ч+SKIP) |
| Захват задачи = открыть сайт, выбрать проект, форма | Голосовое боту за 3 секунды, маршрутизация одним тапом |
| Статус-репорт партнёрам = час работы PM | 📊 «Отчёт империи» — одна кнопка, скопировал в чат |
| Приоритет = что громче горит | 💰 деньги + 🔋 энергия дня вшиты в приоритизацию |
| Onboarding = аккаунты, инвайты, обучение | Вход через Telegram, zero-touch (TMA) |

**Стратегия клина: не миграция, а надстройка.** Фаундер заводит WANDO лично ПОВЕРХ рабочих трекеров (личный командный центр), команда остаётся где была. Потом втягивает 1–2 партнёров через бот и Приёмку. Land-and-expand без switching cost.

## ICP (кто платит)

1. **Соло/дуо-фаундер 2–6 проектов** (агентство + пет-проекты + клиенты) — наш Роч. Боль: «в каком проекте что горит и за что хвататься».
2. **Микрокоманда 2–5 людей** без выделенного PM — Приёмка + роли + ритуалы заменяют PM'а.
3. РУ/СНГ + глобальные инди-фаундеры, живущие в Telegram.

## За что полюбят (уже построено)

- «Твой следующий шаг · вся империя» — единственный экран, который отвечает «что сейчас», а не хранит несделанное.
- Задача голосом из Telegram где угодно; «система никогда не угадывает молча».
- Вандо: ИИ-chief-of-staff с памятью («запомни: работаем по предоплате») и правом молчать.
- Ритуалы: бриф 9:00 → фокус, разбор 18:00 → перенос одним тапом.
- Дофамин: свайпы+хаптика, конфетти, монеты, спекуляр — «инструмент из будущего».
- 📊 Отчёт империи за 1 секунду — и это ещё и вирусный канал (футер «собрано WANDO · cock-pit.com» в каждом отчёте, улетающем в чужие чаты).

## Монетизация

**Модель: freemium-подписка.** (Не lifetime — AI-косты постоянные; не usage — фаундеру нужна предсказуемость.)

| Тир | Цена | Что входит |
|---|---|---|
| **Solo** | $0 | 1 компания, до 3 проектов, бот+TMA, без Вандо-ИИ и отчётов |
| **Founder** | **$19/мес** ($15 годовой) | Империя безлимит, Вандо-ИИ + пульс, ритуалы, 📊 отчёты, память |
| **Team** | **$49/мес** | + до 10 людей, роли/гости, Приёмка с маршрутизацией, командные пуши |

AI-фичи только в платных тирах — они же и есть маржинальный кост.

**Платёжные рельсы (Грузия, Stripe закрыт):**
1. **Telegram Stars** — первично: подписка прямо в TMA/боте, работает из GE, опыт уже есть (NARDUM).
2. **Paddle** (если апелляция ANZH пройдёт) или **Cryptomus/USDT** — для web-чекаута.
3. Годовые планы инвойсом на ИП (TBC) для СНГ-агентств.

## Путь к multi-tenant (тех. минимум, ~2-3 недели)

1. `workspaces` таблица + `workspace_id` на projects/intake/team/tg_links; RLS по membership (SECURITY DEFINER — паттерн уже отработан).
2. Self-serve: `/start` у бота → «создать пространство» → magic-link (tma-auth уже умеет).
3. Метеринг Вандо/пульса по workspace → фича-гейт тиров.
4. Billing: Stars-инвойсы в боте + webhook → `subscriptions`.
5. Изоляция ботов не нужна: один @wando_tasks_bot, маппинг chat→workspace.

## Дистрибуция (в порядке приоритета)

1. **Build in public** — Роч сам ICP: тред «как я управляю 8 проектами из Telegram» + скринкасты. 
2. **Вирусные петли в продукте**: футер отчётов; приглашение партнёра = ссылка бота; «Сделано в WANDO» на публичных дайджестах.
3. Telegram-каналы фаундеров/инди СНГ → потом Product Hunt (после self-serve).
4. AppHub-синергия: WANDO как внутренний инструмент студии → кейс для клиентов студии.

## Чего НЕ делать (антискоуп)

- Не конкурировать с Jira по фичам (кастомные воркфлоу, спринты, эпики) — проиграем и потеряем простоту.
- Не строить свой чат/доски/календарь — Telegram и ICS-фид.
- Не гнаться за enterprise — наш покупатель платит картой/Stars за себя.

## Метрика Севера

**«Сколько фаундеров открыли WANDO ≥5 дней из 7»** (weekly habit) — до неё нет смысла лить трафик.
